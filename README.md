# austral_parts

Base inicial para una app Flutter de inventario.

## Estado actual

Este repositorio contiene un esqueleto minimo para empezar:
- configuracion de dependencias
- app Flutter con pantalla inicial
- prueba basica de widget

## Proximo paso cuando Flutter este disponible

1. Asegura Flutter en PATH.
2. Ejecuta en la raiz del proyecto:

```bash
flutter create .
flutter pub get
flutter run
```

El comando `flutter create .` agregara las carpetas nativas faltantes (android, ios, etc.) usando este esqueleto como base.
